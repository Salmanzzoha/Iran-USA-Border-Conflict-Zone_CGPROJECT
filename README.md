# 🇺🇸 Iran–USA Border Conflict Zone: 2D Computer Graphics Scene

**Computer Graphics Project | Spring 2026**  
**American International University-Bangladesh (AIUB)**

---

### 📋 Project Overview

A dramatic **nighttime military border standoff** simulation built using **OpenGL + GLUT** in C++. The scene features two opposing forces separated by a fortified border with explosions, watchtowers, tanks, jets, and atmospheric effects.
