## 🎯 Overview

**Iran–USA Border Conflict Zone** is a **2D computer graphics scene** simulating a tense nighttime military standoff between two opposing forces. Built entirely from scratch using **OpenGL primitives** in C++, the scene renders a fortified border with full atmospheric and military detail — no textures, no external assets, just geometry and color.

```
🇮🇷  Iranian Side                         American Side  🇺🇸
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Watchtowers · Tanks · Soldiers     Watchtowers · Tanks · Soldiers
  Jet aircraft · Ground forces         Jet aircraft · Ground forces
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
              ⬆️   Fortified Border Wall + Checkpoint   ⬆️
```

**Key OpenGL primitives used:**

```cpp
GL_POINTS       // Stars, debris particles
GL_LINES        // Border lines, spotlight beams
GL_TRIANGLES    // Jets, weapon shapes, explosion shards
GL_QUADS        // Buildings, tanks, wall segments
GL_POLYGON      // Watchtower bodies, soldier silhouettes
glColor3f()     // Scene coloring
glOrtho()       // 2D orthographic projection
```

## ▶️ How to Run

**Prerequisites**

```
✅ Code::Blocks IDE (with MinGW compiler)
✅ FreeGLUT library installed
✅ Windows 10 or 11
```

**Steps**

```bash
# Step 1 — Clone the repository
git clone https://github.com/Salmanzzoha/Computer-Graphics-Project_Spring_2026.git

# Step 2 — Open project in Code::Blocks
#   File → Open → select the .cbp project file

# Step 3 — Link FreeGLUT (if not already configured)
#   Settings → Compiler → Linker Settings
#   Add: -lfreeglut -lopengl32 -lglu32

# Step 4 — Build and Run
#   Press F9  (Build & Run)
```

> 💡 **FreeGLUT setup tip:** Place `freeglut.dll` in the same folder as the compiled `.exe` if you get a missing DLL error on launch.
